HTML

HTML (HyperText Markup Language) is used to structure the content of a web page. Here are 12 different HTML examples and explanations:

1. Heading:


        <h1>This is a Heading 1</h1>

This code creates a top-level heading.

2. Paragraph:


        <p>This is a paragraph.</p>

This code defines a paragraph of text.

3. Link:


        <a href="https://www.example.com">Visit Example.com</a>

This code creates a hyperlink to Example.com.

4. Image:


        <img src="image.jpg" alt="Image Description">

This code inserts an image with an alt text.

5. List:


        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
        </ul>

This code creates an unordered list with items.

6. Form:


        <form>
            <label for="name">Name:</label>
            <input type="text" id="name" name="name">
            <input type="submit" value="Submit">
        </form>

This code creates a simple form with an input field and a submit button.

7. Ordered List:


        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
        </ol>

This code creates an ordered list with items.

8. Anchor Target:


        <a href="#section2">Go to Section 2</a>

This code creates a link that navigates to Section 2 on the same page.

9. HTML Comment:


        <!-- This is an HTML comment -->

This code adds a comment in the HTML code.

10. Input Checkbox:


        <input type="checkbox" id="myCheckbox" name="myCheckbox" value="Yes">

This code creates a checkbox input field.

11. HTML Entity:


        <p>&copy; 2023 My Company</p>

This code uses an HTML entity to display the copyright symbol.

12. Meta Tags:


        <meta name="keywords" content="HTML, CSS, JavaScript">

This code defines meta tags for keywords in the document.

CSS

CSS (Cascading Style Sheets) is used to style and format HTML elements. Here are 12 different CSS examples and explanations:

1. Font Color:


        h2 {
            color: green;
        }

This CSS rule changes the color of all <h2> elements to green.

2. Background Color:


        body {
            background-color: #ccc;
        }

This CSS rule sets the background color of the body to light gray.

3. Text Alignment:


        p {
            text-align: justify;
        }

This CSS rule justifies the text in all <p> elements.

4. Border:


        div {
            border: 2px dashed #f00;
        }

This CSS rule adds a 2px red dashed border around all <div> elements.

5. Font Size:


        p {
            font-size: 16px;
        }

This CSS rule sets the font size of all <p> elements to 16 pixels.

6. Margin and Padding:


        .container {
            margin: 10px;
            padding: 20px;
        }

This CSS rule sets margins to 10px and padding to 20px for elements with class "container."

7. Text Decoration:


        a {
            text-decoration: underline;
        }

This CSS rule adds an underline to all links.

8. Box Shadow:


        div {
            box-shadow: 3px 3px 5px #888888;
        }

This CSS rule adds a box shadow to the div element.

9. Flexbox Layout:


        .container {
            display: flex;
            justify-content: center;
            align-items: center;
        }

This CSS rule creates a centered flexbox layout.

10. CSS Transition:


        button {
            transition: background-color 0.3s ease;
        }

This CSS rule adds a smooth transition effect to the button background color.

11. CSS Animation:


        @keyframes slideIn {
            from {
                transform: translateX(-100%);
            }
            to {
                transform: translateX(0);
            }
        }
        
        .element {
            animation: slideIn 1s ease;
        }

This CSS code defines a slide-in animation for an element.

12. Media Queries:


        @media (max-width: 768px) {
            body {
                font-size: 14px;
            }
        }

This CSS code sets a smaller font size for screens with a width of 768 pixels or less.

JavaScript

JavaScript is used to add interactivity to web pages. Here are 12 different JavaScript examples and explanations:

1. Alert Box:


        alert("Hello, JavaScript!");

This code displays an alert box with the message "Hello, JavaScript!"

2. Console Log:


        console.log("Logged message");

This code logs a message to the browser console.

3. Variables:


        var age = 25;

This code declares a variable named "age" and assigns it the value 25.

4. Function:


        function multiply(a, b) {
            return a * b;
        }

This code defines a function that multiplies two numbers.

5. Event Listener:


        document.querySelector('button').addEventListener('click', function() {
            alert('Button Clicked!');
        });

This code adds a click event listener to a button element that displays an alert when clicked.

6. Conditional Statement:


        var temperature = 30;
        if (temperature > 20) {
            console.log("It's a warm day.");
        } else {
            console.log("It's a cold day.");
        }

This code uses an if-else statement to determine the weather based on the temperature.

7. DOM Manipulation:


        document.getElementById('myElement').innerHTML = 'New content';

This code updates the content of an HTML element with the ID 'myElement'.

8. Local Storage:


        localStorage.setItem('username', 'John');
        var username = localStorage.getItem('username');

This code stores and retrieves data using the browser's local storage.

9. Array Methods:


        var fruits = ['apple', 'banana', 'cherry'];
        fruits.push('orange');

This code adds an element to an array using the push() method.

10. JSON Parsing:


        var jsonString = '{"name": "Alice", "age": 30}';
        var person = JSON.parse(jsonString);

This code parses a JSON string into a JavaScript object.

11. Fetch API:


        fetch('https://api.example.com/data')
            .then(response => response.json())
            .then(data => console.log(data));

This code fetches data from an API using the Fetch API.

12. Error Handling:


        try {
            // Code that may throw an error
        } catch (error) {
            console.error('An error occurred:', error);
        }

This code demonstrates error handling using try and catch.
